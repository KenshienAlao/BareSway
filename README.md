# BareSway

**BareSway** is a very simple, lightweight setup for the Sway Wayland compositor. It installs only the basic tools you need to start using Sway, with no extra or unnecessary programs.

It is meant to be a clean starting point so you can build and customize your own Linux desktop setup later.

### Installation

```bash
git clone https://github.com/KenshienAlao/BareSway.git baresway
cd baresway
chmod +x setup.sh
./setup.sh
```

### What it installs
* `sway` - Wayland compositor
* `foot` - Terminal
* `wofi` - App launcher
* `thunar` - File manager
* `firefox` - Web browser
* `code` - VS Code
